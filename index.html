Fuel where & when you need it

See, update and share fuel depots in seconds.

A rapid field tool to locate, record and update helicopter fuel depots on a live map. It turns scattered notes into a single source of truthfewer calls and spreadsheets, faster decisions, safer operations.

Key features

  • Live Map + GPS: auto-centers on your position.
  • Barrel markers: quick visual count per site.
  • Deposit / Withdraw: log ops with operator + qty.
  • Real-time totals: barrels & liters (1 bbl = 200 L).
  • Precise coords: copy exact lat/lng in one click.
  • History log: timestamped record per operator.
  • Local persistence: browser localStorage (demo).

Tech stack (demo) HTML, CSS, JavaScript, Leaflet (OpenStreetMap tiles), Geolocation API, localStorage. Real deployments include a secure backend, authentication, roles and audit logs.
